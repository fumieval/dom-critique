Click the floating speech-bubble button (bottom-right) to enter comment mode, then click any element on this page.

Introduction

This page exists to help you exercise the library. Try commenting on different kinds of elements: headings, paragraphs, list items, code, even the card below.

"The best UI for code review is one that lives next to the code itself."

Features to try

  • Click an element while in comment mode to attach a comment.
  • Open the sidebar entry to jump to the original element.
  • Edit or delete an existing comment from the sidebar.
  • Press Copy Markdown to copy every comment to the clipboard.
  • Reload the page — comments persist via localStorage.
